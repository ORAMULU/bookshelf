## BookShelf — Style Guide 🎨

### Vibe
**Retro Library meets Modern App** — think old bookstore energy with a clean digital twist.

---

### Colors
```css
--bg-primary: #1a1210      /* very dark brown */
--bg-secondary: #2c1f1a    /* dark brown card */
--bg-tertiary: #3d2b23     /* lighter brown hover */
--accent: #e8956d          /* warm orange */
--accent-soft: #c4713f     /* darker orange */
--text-primary: #f5e6d3    /* warm cream */
--text-secondary: #a89080  /* muted brown */
--border: #4a3428          /* subtle border */
```

---

### Typography
```css
/* Headings — old library feel */
font-family: 'Playfair Display', serif;

/* Body — clean and readable */
font-family: 'DM Sans', sans-serif;

/* Import both from Google Fonts */
@import url('https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;700&family=DM+Sans:wght@400;500;600&display=swap');
```

---

### Spacing
```css
--padding-page: 24px
--padding-card: 20px
--border-radius: 12px
--gap: 16px
```

---

### Component Hints

**Cards** — warm brown background, left orange border accent, subtle shadow

**Buttons** — orange background, cream text, rounded

**Inputs** — dark background, warm border, cream text, orange focus outline

**Nav** — fixed bottom, dark brown background, orange active indicator

**Avatar/Initial circle** — orange background, dark text, round

---

### Status Colors
```css
Reading     → #e8956d  /* orange */
Completed   → #6db86d  /* green */
Want to Read → #6d9be8 /* blue */
```
